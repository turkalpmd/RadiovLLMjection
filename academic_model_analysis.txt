ACADEMIC MODEL ANALYSIS REPORT
==============================

EXECUTIVE SUMMARY
--------------------
This comprehensive analysis evaluates 10 multimodal large language models across 5 test scenarios using 1048 medical images across 2 diagnostic categories.

METHODOLOGY
------------
Models were tested using ground truth classification, injection attacks, and immune defense mechanisms. Performance metrics include accuracy rates, deception rates, and statistical significance tests.

KEY RESULTS
------------
• Dataset contains 6,000 test instances across 10 multimodal models
• Average GT accuracy: 82.9% (SD: 12.8%)
• Average injection attack success: 79.4%
• Most vulnerable model: anthropic_claude_sonnet_4_5
• Most robust model: google_gemini_3_pro_preview
• Data completeness: 59.8% average across all test types
